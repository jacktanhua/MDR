The mdr-readability is a Python package for computing the MDR (Multi-Dimensional Readability) score and 42 linguistic features from English text. 
MDR is a regression-based readability index that combines lexical, syntactic, and semantic features to predict text difficulty. 
The package is available at https://pypi.org/project/mdr-readability/ and can be installed via pip install mdr-readability.
